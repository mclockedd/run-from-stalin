Drop as many audio files here as you like. The game auto-discovers them — no
config to edit. A file's CATEGORY is decided by how its name STARTS:

  lobby...   -> lobby music pool   (e.g. lobby1.mp3, lobby-anthem.mp3)
  game...    -> game music pool    (e.g. game1.mp3, game-chase.mp3)
  kill...    -> kill sound pool    (e.g. kill.mp3, kill2.mp3)

How they play:
- Each time you ENTER THE LOBBY, one random track from the lobby pool plays (loops).
- Each time a ROUND STARTS, one random track from the game pool plays (loops).
- Each time Stalin CATCHES someone, one random clip from the kill pool plays once.

Formats: .mp3 (safest, works everywhere), .ogg, or .wav.

Notes:
- Any number of files per category is fine — more = more variety.
- If a pool is empty, that sound is simply silent; the game still works.
- Music starts after you click "Create Party" / "Join" (browsers block audio
  until you interact with the page). The speaker button (bottom-right) mutes.
